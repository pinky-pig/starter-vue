# FlipCard

demo

## 介绍

一个可翻转的卡片。

### 安装

```bash
pnpm i @arvin/materials
```

### 使用

```js
import { FlipCard } from '@arvin/materials'
```

```vue
<template>
  <FlipCard />
</template>
```

## API

| 参数        | 说明     | 类型   | 默认值  |
| ----------- | -------- | ------ | ------- | ------ |
| align       | 对其方式 | 'left' | 'right' | 'left' |
| rotateAngle | 倾斜角度 | number | -5      |
